# Arutech Consultancy Services LLP — Website v2.0

Enterprise-grade website for **Arutech Consultancy Services LLP**.

> "Building Technology for a Self-Sufficient India."

---

## Tech Stack
- **Next.js 15** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS**
- **Lucide React**
- Clash Display + Satoshi fonts (via Fontshare)

## Run Locally

```bash
npm install
npm run dev
# → http://localhost:3000
```

## Deploy on Vercel

```bash
npm i -g vercel
vercel --prod
```

Or connect GitHub repo to Vercel dashboard.

## Pages
- `/` — Home (Hero, Services, Products, Why Arutech, Tech Stack, CTA)
- `/services` — Detailed services with capabilities
- `/products` — InstaChat, Arutech AI, roadmap
- `/about` — Mission, values, journey
- `/contact` — Contact form + details
